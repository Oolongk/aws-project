Thực hiện khởi tạo bảng lưu trữ trên Amazon DynamoDB để chứa dữ liệu Meta của các vị Tướng.

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Truy cập DynamoDB | Trên thanh tìm kiếm AWS Console, gõ **DynamoDB** > Chọn **Tables** ở menu trái > Nhấn **Create table**. |
| 2 | Cấu hình Table Name | Table name: `dota2_hero_metrics`. |
| 3 | Cấu hình Khóa chính | Partition key (Khóa phân vùng): Gõ `hero_id` và chọn kiểu dữ liệu là **String (S)**. Sort key là `match_date` và chọn kiểu dữ liệu **String (S)**. |
| 4 | Cấu hình Settings | Tại mục *Table settings*, tích chọn **Customize settings** để cấu hình tối ưu chi phí. |
| 5 | Tối ưu Capacity Mode | Tại mục *Read/write capacity mode*, tích chọn **On-demand** (Pay-per-request). |
| 6 | Hoàn tất | Cuộn xuống cuối trang và nhấn nút **Create table** màu cam. |
