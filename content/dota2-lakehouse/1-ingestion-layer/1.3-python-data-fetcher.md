Chúng ta sẽ viết một Script Python để đóng vai trò Ingestion Bot: gọi OpenDota API, lấy 100 trận đấu chuyên nghiệp mới nhất và ném cục Payload JSON đó lên S3.

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Chuẩn bị môi trường | Mở Terminal (Command Prompt), cài đặt thư viện AWS: `pip install boto3 requests` |
| 2 | Nạp chìa khóa IAM | Gõ lệnh `aws configure`. Lần lượt dán Access Key, Secret Key vừa lấy ở Bài 1.2. Chọn Region là `ap-southeast-1`. |
| 3 | Viết mã nguồn | Tạo một file tên `dota2_ingestion.py` và dán đoạn code bên dưới vào. (Nhớ sửa tên `BUCKET_NAME`). |
| 4 | Chạy Script | Gõ lệnh `python dota2_ingestion.py` trên Terminal. |
| 5 | Kiểm duyệt (Validate) | Truy cập AWS Console > **S3** > Mở Raw Bucket. Bạn sẽ thấy một cấu trúc thư mục mới được tạo tự động chứa file JSON. |

Mã nguồn `dota2_ingestion.py`:

```python
import requests
import boto3
import json
from datetime import datetime

# Cấu hình API và S3
API_URL = "[https://api.opendota.com/api/proMatches](https://api.opendota.com/api/proMatches)"
BUCKET_NAME = "dota2-raw-zone-tencuaban" # Thay bằng tên S3 Raw Bucket của bạn
PREFIX = "raw_matches"

def fetch_and_upload():
    print("Đang gọi OpenDota API...")
    response = requests.get(API_URL)
    
    if response.status_code == 200:
        matches = response.json()
        
        # Tạo đường dẫn theo chuẩn Apache Hive Partitioning (dt=YYYY-MM-DD)
        date_str = datetime.now().strftime("%Y-%m-%d")
        file_name = f"matches_{int(datetime.now().timestamp())}.json"
        s3_key = f"{PREFIX}/dt={date_str}/{file_name}"
        
        # PutObject lên Amazon S3
        s3 = boto3.client('s3')
        s3.put_object(
            Bucket=BUCKET_NAME,
            Key=s3_key,
            Body=json.dumps(matches),
            ContentType='application/json'
        )
        print(f"✅ Thành công! Đã đẩy {len(matches)} trận đấu lên S3 tại: {s3_key}")
    else:
        print("❌ Lỗi API:", response.status_code)

if __name__ == "__main__":
    fetch_and_upload()
```
