Đây là linh hồn của hệ thống. Chúng ta sẽ tạo một Job ETL trên AWS Glue để đọc file JSON, làm phẳng mảng cầu thủ và ghi ra Parquet.

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Mở AWS Glue | Vào Console, tìm **AWS Glue** > Chọn **ETL jobs** (menu trái). |
| 2 | Khởi tạo Job | Chọn tùy chọn **Script editor** > Chọn **Spark** > Nhấn **Create script**. |
| 3 | Cấu hình Job Details | Chuyển sang tab **Job details**. Đặt tên: `dota2_etl_job`. Mục IAM Role, chọn `Glue_Dota2_Processing_Role` (vừa tạo ở bước 2.2). |
| 4 | Tối ưu chi phí | Trong phần *Job bookmark*, chọn Enable. Phần *Number of workers*, giảm từ 10 xuống **2** (Vì chúng ta đang dùng dữ liệu nhỏ, không cần máy quá to). |
| 5 | Chèn Code | Chuyển lại tab **Script**, dán đoạn code PySpark bên dưới vào. Nhớ thay thế các biến `RAW_PATH` và `PROCESSED_PATH` bằng tên Bucket của bạn. |
| 6 | Lưu và Chạy | Nhấn **Save** > Nhấn **Run**. Đợi khoảng 2-3 phút, sau đó kiểm tra S3 Processed Bucket để xem thành quả. |

**Đoạn mã PySpark hoàn chỉnh:**

```python
import sys
from awsglue.transforms import *
from awsglue.utils import getResolvedOptions
from pyspark.context import SparkContext
from awsglue.context import GlueContext
from awsglue.job import Job
from pyspark.sql.functions import explode, col, from_unixtime, to_date

sc = SparkContext.getOrCreate()
glueContext = GlueContext(sc)
spark = glueContext.spark_session

RAW_PATH = "s3://dota2-raw-zone-tencuaban/raw_matches/"
PROCESSED_PATH = "s3://dota2-processed-zone-tencuaban/matches_parquet/"

# 1. Đọc dữ liệu JSON thô
print("Bắt đầu đọc dữ liệu từ S3 Raw...")
df = spark.read.json(RAW_PATH)

# 2. Xử lý dữ liệu (Flattening)
# Bóc tách mảng participants thành từng dòng riêng biệt
df_exploded = df.select(
    col("match_id"),
    to_date(from_unixtime(col("start_time"))).alias("match_date"),
    explode(col("players")).alias("player")
)

# Rút trích các trường cần thiết từ object player
df_final = df_exploded.select(
    "match_id",
    "match_date",
    col("player.account_id").alias("account_id"),
    col("player.hero_id").alias("hero_id"),
    col("player.win").alias("is_win")
)

# 3. Ghi dữ liệu ra định dạng Parquet nén Snappy
print("Bắt đầu ghi dữ liệu ra định dạng Parquet...")
df_final.write.mode("append") \
    .partitionBy("match_date") \
    .parquet(PROCESSED_PATH, compression="snappy")

print("Hoàn tất Job ETL!")
```
