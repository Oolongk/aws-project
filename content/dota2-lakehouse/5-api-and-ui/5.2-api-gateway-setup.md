Chúng ta tiến hành thiết lập một cổng API Gateway để chuyển đổi hàm Lambda Backend thành một địa chỉ URL công khai (Endpoint) trên mạng Internet.

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Mở API Gateway | Tìm kiếm dịch vụ **API Gateway** trên AWS Console. |
| 2 | Chọn loại API | Tìm đến mục **HTTP API** > Nhấn nút **Build** màu cam (HTTP API rẻ và nhanh hơn REST API gấp nhiều lần). |
| 3 | Cấu hình Integrations | Nhấn *Add integration* > Chọn **Lambda**. Chọn Region `ap-southeast-1` > Ô *Lambda function*, chọn `dota2-backend-api` > API name điền: `dota2-public-api` > Nhấn **Next**. |
| 4 | Định tuyến Routes | Configure routes: Method chọn **GET**. Path điền: `/metrics`. Target giữ nguyên hàm Lambda. Nhấn **Next**. |
| 5 | Giữ nguyên Stage | Mục Stages giữ mặc định là `$default` (bật tính năng Auto-deploy). Nhấn **Next**. |
| 6 | Hoàn tất | Nhấn **Create**. Hệ thống sẽ cấp cho bạn một chuỗi đường dẫn nằm ở ô **Invoke URL** dạng `https://xxxx.execute-api.ap-southeast-1.amazonaws.com`. |
| 7 | Thử nghiệm API | Copy đường link Invoke URL dán lên trình duyệt và gõ thêm đuôi: `/metrics?hero_id=74`. Nếu màn hình nhả ra cục dữ liệu JSON dạng `{"hero_id": 74, "win_rate": "52.4"}` nghĩa là bạn đã thành công 100%! |
