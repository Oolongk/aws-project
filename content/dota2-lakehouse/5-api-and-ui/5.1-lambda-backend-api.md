Chúng ta sẽ xây dựng một hàm Lambda đóng vai trò là API Backend chịu trách nhiệm tiếp nhận tham số `hero_id` gửi lên từ trình duyệt, truy vấn vào DynamoDB và trả về kết quả JSON.

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Truy cập AWS Console | Tìm kiếm dịch vụ **Lambda** trên thanh công cụ. |
| 2 | Khởi tạo Hàm | Chọn **Functions** ở menu trái > Nhấn nút **Create function** màu cam. |
| 3 | Cấu hình cơ bản | Chọn *Author from scratch*. Function name: `dota2-backend-api`. Runtime: Chọn **Python 3.9** (hoặc bản mới hơn). |
| 4 | Cấu hình Permissions | Mở rộng mục *Change default execution role* > Chọn **Create a new role with basic Lambda permissions** (AWS sẽ tự tạo một Role cơ bản bảo mật). Nhấn **Create function**. |
| 5 | Gắn quyền đọc DynamoDB | Sang tab *Configuration* > Mục *Permissions* > Click vào đường link *Role name* màu xanh. Hệ thống sẽ nhảy sang tab IAM. |
| 6 | Thêm Inline Policy | Nhấn **Add permissions** > **Create inline policy** > Chọn tab **JSON** và dán đoạn mã bên dưới vào (Nhớ sửa `YOUR_ACCOUNT_ID`) > Đặt tên policy: `DynamoDB_Read_Dota2_Metrics` > Nhấn **Create policy**. |
| 7 | Viết mã nguồn API | Quay lại tab Lambda Function > Kéo xuống phần Code editor, xóa sạch code mặc định và dán đoạn mã Backend Python bên dưới vào > Nhấn nút **Deploy** màu cam. |

Mã JSON Policy cấp quyền đọc cho Lambda:
```json
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Effect": "Allow",
            "Action": [
                "dynamodb:GetItem"
            ],
            "Resource": "arn:aws:dynamodb:ap-southeast-1:YOUR_ACCOUNT_ID:table/dota2_hero_metrics"
        }
    ]
}
```

Mã JSON Policy cấp quyền đọc cho Lambda:
```python
import json
import boto3

DYNAMODB_TABLE = "dota2_hero_metrics"

def lambda_handler(event, context):
    print("Received event:", json.dumps(event))
    
    # 1. Trích xuất tham số hero_id từ đường dẫn API Gateway (Query String hoặc Path Parameter)
    query_params = event.get('queryStringParameters') or {}
    hero_id = query_params.get('hero_id')
    
    if not hero_id:
        return {
            'statusCode': 400,
            'headers': {
                'Access-Control-Allow-Origin': '*', # Bật CORS chống lỗi chặn Browser
                'Content-Type': 'application/json'
            },
            'body': json.dumps({'error': 'Tham số hero_id là bắt buộc!'})
        }
    
    # 2. Truy vấn dữ liệu trực tiếp từ Amazon DynamoDB theo Key Lookups
    dynamodb = boto3.resource('dynamodb', region_name='ap-southeast-1')
    table = dynamodb.Table(DYNAMODB_TABLE)
    
    response = table.get_item(Key={'hero_id': int(hero_id)})
    item = response.get('Item')
    
    if not item:
        return {
            'statusCode': 404,
            'headers': {'Access-Control-Allow-Origin': '*', 'Content-Type': 'application/json'},
            'body': json.dumps({'message': f'Không tìm thấy dữ liệu cho Hero ID {hero_id}'})
        }
        
    # 3. Trả kết quả JSON sạch về cho Frontend
    return {
        'statusCode': 200,
        'headers': {
            'Access-Control-Allow-Origin': '*',
            'Content-Type': 'application/json'
        },
        'body': json.dumps(item)
    }
```
