Chúng ta sẽ đưa mã nguồn giao diện Website tĩnh (HTML/JS) lên một S3 Bucket đặc biệt được kích hoạt tính năng phát sóng Website (Static Website Hosting).

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Tạo Bucket giao diện | Vào **S3 Console**, tạo một bucket mới tên: `dota2-frontend-site-tencuaban`. |
| 2 | Tắt Block Public Access | Vì đây là trang web công khai cho mọi người vào xem, tại mục *Block Public Access*, bạn **BỎ TÍCH** ô *Block all public access*. Nhấn tạo Bucket. |
| 3 | Cấu hình Web Hosting | Mở Bucket vừa tạo > Chọn tab **Properties** > Cuộn xuống cuối cùng tìm mục **Static website hosting** > Chọn **Edit** > Chọn **Enable**. |
| 4 | Điền file mặc định | Mục *Index document*, nhập `index.html`. Nhấn **Save changes**. Hệ thống nhả ra một đường link trang web ở dưới cùng mục này. |
| 5 | Tạo mã nguồn cục bộ | Tại máy tính của bạn, tạo một file tên là `index.html` và dán đoạn mã nguồn HTML/JS ở khung bên dưới vào. (Nhớ thay thế biến `API_GATEWAY_URL` bằng đường link Invoke URL bạn vừa lấy ở bài 5.2). |
| 6 | Đẩy file lên S3 | Quay lại S3 Console > Tab **Objects** > Nhấn **Upload** và tải file `index.html` lên. |
| 7 | Mở quyền Bucket Policy | Chuyển sang tab **Permissions** > Mục **Bucket policy** > Nhấn **Edit** và dán đoạn mã cấu hình JSON mở quyền đọc ở bên dưới vào > Nhấn **Save**. |

Mã JSON cho Bucket Policy (Nhớ thay `dota2-frontend-site-tencuaban` bằng tên bucket của bạn):
```json
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "PublicReadGetObject",
            "Effect": "Allow",
            "Principal": "*",
            "Action": "s3:GetObject",
            "Resource": "arn:aws:s3:::dota2-frontend-site-tencuaban/*"
        }
    ]
}
