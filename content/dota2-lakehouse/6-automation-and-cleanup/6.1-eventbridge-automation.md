Chúng ta sẽ thực hiện 2 tác vụ: Đưa script lấy dữ liệu lên AWS Lambda, và thiết lập đồng hồ báo thức (EventBridge Schedule) để nó tự chạy vào lúc 2 giờ sáng mỗi ngày.

**Phần 1: Tạo Lambda Ingestion (Thay thế cho máy tính cá nhân)**

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Tạo Hàm Lambda | Truy cập **Lambda** > Nhấn **Create function**. Đặt tên: `dota2-auto-ingestion`. Runtime: **Python 3.9**. |
| 2 | Cấp quyền S3 | Tương tự bài 5.1, vào tab **Configuration** > **Permissions** > Mở Role và thêm Inline Policy cho phép `s3:PutObject` vào bucket Raw. |
| 3 | Chèn mã nguồn | Dán đoạn mã Python bên dưới vào Code Editor. Thay `YOUR_RAW_BUCKET` bằng tên bucket của bạn. Nhấn **Deploy**. |
| 4 | Cấu hình Timeout | Sang tab **Configuration** > **General configuration** > **Edit**. Tăng Timeout từ 3 giây lên **1 phút**. Nhấn **Save**. |

**Mã nguồn `dota2-auto-ingestion` (Dùng thư viện `urllib` mặc định thay vì `requests` để không cần cài Layer):**
```python
import urllib.request
import boto3
import json
from datetime import datetime

API_URL = "[https://api.opendota.com/api/proMatches](https://api.opendota.com/api/proMatches)"
BUCKET_NAME = "YOUR_RAW_BUCKET"
PREFIX = "raw_matches"

def lambda_handler(event, context):
    print("Bắt đầu lấy dữ liệu từ OpenDota API...")
    req = urllib.request.Request(API_URL, headers={'User-Agent': 'Mozilla/5.0'})
    
    with urllib.request.urlopen(req) as response:
        if response.status == 200:
            matches = json.loads(response.read().decode())
            date_str = datetime.now().strftime("%Y-%m-%d")
            file_name = f"matches_{int(datetime.now().timestamp())}.json"
            s3_key = f"{PREFIX}/dt={date_str}/{file_name}"
            
            s3 = boto3.client('s3')
            s3.put_object(Bucket=BUCKET_NAME, Key=s3_key, Body=json.dumps(matches), ContentType='application/json')
            
            return {"status": 200, "message": f"Đã đẩy {len(matches)} trận lên S3"}
        else:
            return {"status": response.status, "message": "Lỗi gọi API"}
