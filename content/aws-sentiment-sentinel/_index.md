# Social Media Sentiment Sentinel

Chào mừng bạn đến với dự án xây dựng hệ thống phân tích cảm xúc bình luận YouTube tự động (Serverless).

Trong series này, chúng ta sẽ giải quyết bài toán Big Data thực tế:
1.  Thu thập dữ liệu tự động từ YouTube.
2.  Sử dụng Generative AI (**Claude 3**) để hiểu ngữ cảnh và sự châm biếm.
3.  Vượt qua giới hạn Rate Limit của tài khoản AWS Academy.
4.  Trực quan hóa dữ liệu trên QuickSight.
