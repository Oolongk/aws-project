# AWS FCJ Projects Portfolio

Các project trong portfolio này tập trung vào việc xây dựng hệ thống cloud thực tế trên AWS, bao gồm Data Lakehouse, Big Data Processing, Serverless Data Pipeline, Real-time Streaming, Analytics Dashboard và AI Recommendation System.

## Mục tiêu của các Workshop

Workshop này được xây dựng nhằm thể hiện khả năng thiết kế và triển khai hệ thống dữ liệu trên AWS theo nhiều mô hình khác nhau:

- Batch Data Pipeline
- Real-time Streaming Pipeline
- Data Lakehouse Architecture
- Serverless Data Processing
- Big Data Analytics
- AI-powered Recommendation System
- Dashboard & Sentiment Analytics
- Automation, Monitoring và Cost Optimization

## Danh sách project

| STT | Project | Mô tả ngắn |
|---|---|---|---|
| 1 | Dota 2 Meta Analytics Data Lakehouse | Xử lý Big Data eSports với AWS Glue PySpark, Amazon S3 Data Lake và Amazon Athena để phân tích meta game Dota 2. |
| 2 | Quantamental Stock Screener Time-Series Pipeline | Sàng lọc cổ phiếu và backfill dữ liệu tài chính lịch sử bằng kiến trúc ELT siêu nhẹ với AWS Glue Python Shell. |
| 3 | IoT Fleet Monitoring System | Data pipeline theo thời gian thực dùng AWS IoT Core và Amazon SQS để giám sát trạng thái đội xe tải. |
| 4 | Clickstream Analytics & AI Recommender | Thu thập hành vi người dùng, phân tích clickstream và tích hợp Amazon Personalize để gợi ý nội dung bằng AI. |
| 5 | YouTube Sentinel & Dashboard | Hệ thống serverless tự động phân tích cảm xúc bình luận YouTube và hiển thị kết quả trên dashboard. |
