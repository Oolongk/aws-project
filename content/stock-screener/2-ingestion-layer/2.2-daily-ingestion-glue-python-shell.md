Daily Ingestion lấy danh sách S&P 500 từ Wikipedia, lấy dữ liệu từ Yahoo Finance qua `yfinance`, sau đó ghi ba Raw dataset vào Amazon S3:

````text
us_prices_raw.parquet
us_company_info.parquet
us_financials_raw.parquet
````

Khác với Backfill chạy local, bài này tạo AWS Glue Python Shell Job và **dán trực tiếp source code vào Glue Script editor**. Không cần tự tạo thư mục `glue-scripts/` hoặc upload file `daily_ingestion.py` lên S3 bằng tay.

## Chuẩn bị trước khi tạo Job

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Kiểm tra Raw Bucket | Bucket đang dùng là `stock-screener-raw-khanh` |
| 2 | Kiểm tra IAM Role | Role `AWSGlueServiceRole-StockScreener` phải có quyền ghi vào `raw_us_market/*` |
| 3 | Kiểm tra Internet access | Job cần gọi Wikipedia, Yahoo Finance và PyPI để cài package bổ sung |
| 4 | Chuẩn bị source code | Copy toàn bộ code `daily_ingestion.py` ở cuối bài |
| 5 | Không hard-code credential | `boto3.client("s3")` tự dùng temporary credential của Glue Role |

> **Tư duy kiến trúc sư – direct editor.** Glue Studio cho phép tạo Python Shell script mới và viết code trực tiếp trong Script editor. Bạn không cần tự upload file script lên S3. AWS Glue vẫn quản lý Script Location cho Job ở tầng dịch vụ, nhưng không cần thao tác thủ công với một thư mục `glue-scripts/`.

## Kiểm tra kết quả trên Amazon S3

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Amazon S3 → `stock-screener-raw-khanh` | Mở Raw Bucket |
| 2 | Mở `raw_us_market/` | Xem các partition theo năm/tháng/ngày |
| 3 | Tìm `us_prices_raw.parquet` | Partition dùng ngày giao dịch mới nhất |
| 4 | Tìm `us_company_info.parquet` | Partition dùng ngày Job chạy |
| 5 | Tìm `us_financials_raw.parquet` | Partition dùng ngày Job chạy |
| 6 | Mở CloudWatch Logs của Job | Đối chiếu log `Đã lưu S3` với object thực tế |


> **Tư duy kiến trúc sư – schema parity.** Daily Ingestion ép toàn bộ metric về `float64` và các trường ngày về `string` trước khi ghi Parquet. Backfill cũng áp dụng cùng quy tắc. Đây là lớp bảo vệ chính chống schema drift, đặc biệt khi `yfinance` thay đổi kiểu cột hoặc thiếu `Adj Close`, `Dividends` hay `Stock Splits`.

## Full source: `daily_ingestion.py`

````python
import io
import time
import traceback
from datetime import datetime

import boto3
import pandas as pd
import requests
import yfinance as yf

# ==========================================
# CẤU HÌNH CƠ BẢN
# ==========================================
BUCKET_NAME = "stock-screener-raw-khanh"
RAW_PREFIX = "raw_us_market"

# Đổi thành None khi muốn chạy thật toàn bộ S&P 500
LIMIT_TICKERS_FOR_TEST = None

s3 = boto3.client("s3")

# ==========================================
# HÀM LẤY TICKER (GIỐNG BACKFILL)
# ==========================================
def get_sp500_tickers():
    url = "https://en.wikipedia.org/wiki/List_of_S%26P_500_companies"
    headers = {
        "User-Agent": (
            "Mozilla/5.0 (Windows NT 10.0; Win64; x64) "
            "AppleWebKit/537.36 Chrome/120.0 Safari/537.36"
        )
    }
    response = requests.get(url, headers=headers, timeout=30)
    response.raise_for_status()
    df_sp500 = pd.read_html(response.text)[0]
    return [ticker.replace(".", "-") for ticker in df_sp500["Symbol"].tolist()]


# ==========================================
# HÀM XỬ LÝ PARQUET (GIỐNG BACKFILL 100%)
# ==========================================
def to_parquet_bytes(df: pd.DataFrame) -> bytes:
    """
    Chuyển DataFrame thành Parquet tương thích AWS Glue.
    """
    output_df = df.copy()

    for column_name in output_df.columns:
        if pd.api.types.is_datetime64_any_dtype(output_df[column_name]):
            date_series = output_df[column_name]

            if isinstance(date_series.dtype, pd.DatetimeTZDtype):
                date_series = date_series.dt.tz_localize(None)

            # Phân tách format theo tên cột giống backfill
            if column_name in ["price_date", "report_date", "snapshot_date"]:
                output_df[column_name] = date_series.dt.strftime("%Y-%m-%d")
            else:
                output_df[column_name] = date_series.dt.strftime(
                    "%Y-%m-%d %H:%M:%S"
                )

    buf = io.BytesIO()
    output_df.to_parquet(
        buf,
        index=False,
        compression="snappy",
        engine="pyarrow",
        version="1.0"
    )
    return buf.getvalue()


def put_df(df: pd.DataFrame, key: str) -> tuple:
    """Chuẩn hóa kiểu dữ liệu rồi đẩy Parquet lên S3."""
    if df.empty:
        print(f"⚠️ Bỏ qua file rỗng: {key}")
        return 0, 0

    output_df = df.copy()

    # Ép Float64 cho metric để tránh lỗi Integer vs Double trong Spark.
    numeric_columns = []
    if "us_prices_raw" in key:
        numeric_columns = [
            "open",
            "high",
            "low",
            "close",
            "adj_close",
            "volume",
            "dividends",
            "stock_splits"
        ]
    elif "us_company_info" in key:
        numeric_columns = [
            "current_market_cap",
            "shares_outstanding",
            "enterprise_value",
            "float_shares",
            "beta",
            "trailing_pe",
            "return_on_equity",
            "debt_to_equity",
            "earnings_growth",
            "revenue_growth"
        ]
    elif "us_financials_raw" in key:
        numeric_columns = [
            "revenue",
            "net_income",
            "shareholders_equity",
            "total_assets",
            "total_debt",
            "operating_cash_flow",
            "capital_expenditure",
            "free_cash_flow",
            "stock_repurchased",
            "stock_issued"
        ]

    for col in numeric_columns:
        if col in output_df.columns:
            output_df[col] = (
                pd.to_numeric(output_df[col], errors="coerce")
                .astype("float64")
            )

    # Ép trường ngày thành string để Spark/Glue đọc schema ổn định.
    for date_col in [
        "price_date",
        "report_date",
        "snapshot_date",
        "ingested_at"
    ]:
        if date_col in output_df.columns:
            output_df[date_col] = output_df[date_col].astype("string")

    s3.put_object(
        Bucket=BUCKET_NAME,
        Key=key,
        Body=to_parquet_bytes(output_df),
        ContentType="application/octet-stream"
    )
    print(f"✅ Đã lưu S3: {key} | Số dòng: {len(output_df)}")
    return len(output_df), len(output_df.columns)


# ==========================================
# HÀM CRAWL DỮ LIỆU MỖI NGÀY
# ==========================================
def run_yfinance_full_ingestion():
    now = datetime.now()

    # Company Info và Financials lưu theo ngày pipeline chạy.
    yyyy = now.strftime("%Y")
    mm = now.strftime("%m")
    dd = now.strftime("%d")
    current_snapshot_date = now.strftime("%Y-%m-%d")
    ingested_at = now.strftime("%Y-%m-%d %H:%M:%S")

    raw_prefix_today = f"{RAW_PREFIX}/{yyyy}/{mm}/{dd}"

    print("=== BẮT ĐẦU DAILY STOCK INGESTION ===")
    print(f"Snapshot date: {current_snapshot_date}")

    all_tickers = get_sp500_tickers()
    if LIMIT_TICKERS_FOR_TEST is not None:
        all_tickers = all_tickers[:LIMIT_TICKERS_FOR_TEST]

    all_prices, all_financials, all_info = [], [], []

    def get_val(df, possible_rows, report_date):
        if df is None or df.empty:
            return None

        for row_name in possible_rows:
            if row_name in df.index:
                try:
                    return df.loc[row_name, report_date]
                except Exception:
                    return None

        return None

    for i, t in enumerate(all_tickers, 1):
        print(f"\n🔄 Đang xử lý mã: {t} ({i}/{len(all_tickers)})")

        try:
            stock = yf.Ticker(t)

            # ---------------------------------------------------------
            # 1. LẤY GIÁ PHIÊN MỚI NHẤT
            # ---------------------------------------------------------
            try:
                # Lấy 5 ngày để vẫn có phiên gần nhất khi chạy cuối tuần.
                df_price = stock.history(period="5d", auto_adjust=False)

                if not df_price.empty:
                    df_price = df_price.reset_index()

                    if "Date" in df_price.columns:
                        df_price = df_price.rename(
                            columns={"Date": "price_date"}
                        )

                    if "Datetime" in df_price.columns:
                        df_price = df_price.rename(
                            columns={"Datetime": "price_date"}
                        )

                    df_price["price_date"] = (
                        pd.to_datetime(df_price["price_date"])
                        .dt.tz_localize(None)
                    )

                    rename_map = {
                        "Open": "open",
                        "High": "high",
                        "Low": "low",
                        "Close": "close",
                        "Adj Close": "adj_close",
                        "Volume": "volume",
                        "Dividends": "dividends",
                        "Stock Splits": "stock_splits"
                    }
                    df_price = df_price.rename(columns=rename_map)

                    # Khi yfinance không trả Adj Close, dùng Close làm fallback.
                    if (
                        "adj_close" not in df_price.columns
                        and "close" in df_price.columns
                    ):
                        df_price["adj_close"] = df_price["close"]

                    df_price["ticker"] = t
                    df_price["snapshot_date"] = current_snapshot_date
                    df_price["ingested_at"] = ingested_at

                    # Chỉ lưu phiên giao dịch mới nhất của ticker.
                    latest_date = df_price["price_date"].max()
                    df_price_latest = df_price[
                        df_price["price_date"] == latest_date
                    ].copy()

                    expected_cols = [
                        "price_date",
                        "ticker",
                        "open",
                        "high",
                        "low",
                        "close",
                        "adj_close",
                        "volume",
                        "dividends",
                        "stock_splits",
                        "snapshot_date",
                        "ingested_at"
                    ]
                    final_cols = [
                        c for c in expected_cols
                        if c in df_price_latest.columns
                    ]
                    df_price_latest = df_price_latest[final_cols]

                    if not df_price_latest.empty:
                        all_prices.append(df_price_latest)
                        print(
                            "  ✅ Giá mới nhất: "
                            f"{latest_date.strftime('%Y-%m-%d')}"
                        )
                else:
                    print("  ⚠️ Price data rỗng.")

            except Exception as e:
                print(f"  ⚠️ Lỗi Prices: {e}")

            # ---------------------------------------------------------
            # 2. LẤY COMPANY INFO
            # ---------------------------------------------------------
            try:
                info = stock.info or {}
            except Exception:
                info = {}

            try:
                fast_info = stock.fast_info or {}
                market_cap = (
                    fast_info.get("marketCap")
                    or info.get("marketCap", 0)
                )
            except Exception:
                fast_info = {}
                market_cap = info.get("marketCap", 0)

            if market_cap:
                try:
                    info_dict = {
                        "ticker": t,
                        "snapshot_date": current_snapshot_date,
                        "company_name": (
                            info.get("shortName")
                            or info.get("longName")
                            or t
                        ),
                        "exchange": info.get("exchange"),
                        "sector": info.get("sector"),
                        "industry": info.get("industry"),
                        "country": info.get("country"),
                        "currency": info.get("currency"),
                        "website": info.get("website"),
                        "current_market_cap": market_cap,
                        "shares_outstanding": info.get("sharesOutstanding"),
                        "enterprise_value": info.get("enterpriseValue"),
                        "float_shares": info.get("floatShares"),
                        "beta": info.get("beta"),
                        "trailing_pe": info.get("trailingPE"),
                        "return_on_equity": info.get("returnOnEquity"),
                        "debt_to_equity": info.get("debtToEquity"),
                        "earnings_growth": info.get("earningsGrowth"),
                        "revenue_growth": info.get("revenueGrowth"),
                        "ingested_at": ingested_at
                    }
                    all_info.append(info_dict)
                    print("  ✅ Company info")

                except Exception as e:
                    print(f"  ⚠️ Lỗi Company Info: {e}")

            # ---------------------------------------------------------
            # 3. LẤY FINANCIALS
            # ---------------------------------------------------------
            try:
                income = stock.quarterly_income_stmt
                balance = stock.quarterly_balance_sheet
                cashflow = stock.quarterly_cashflow
                period_type = "quarterly"

                # Fallback sang annual nếu quarterly không có dữ liệu.
                if income.empty and balance.empty and cashflow.empty:
                    income = stock.income_stmt
                    balance = stock.balance_sheet
                    cashflow = stock.cashflow
                    period_type = "annual"

                report_dates = set()

                for statement in [income, balance, cashflow]:
                    if statement is not None and not statement.empty:
                        report_dates.update(statement.columns.tolist())

                for report_date in report_dates:
                    report_date_str = pd.Timestamp(
                        report_date
                    ).strftime("%Y-%m-%d")

                    financial_dict = {
                        "ticker": t,
                        "report_date": report_date_str,
                        "snapshot_date": current_snapshot_date,
                        "period_type": period_type,
                        "revenue": get_val(
                            income,
                            ["Total Revenue", "Operating Revenue"],
                            report_date
                        ),
                        "net_income": get_val(
                            income,
                            [
                                "Net Income",
                                "Net Income Common Stockholders"
                            ],
                            report_date
                        ),
                        "shareholders_equity": get_val(
                            balance,
                            [
                                "Stockholders Equity",
                                "Stockholders' Equity",
                                "Common Stock Equity"
                            ],
                            report_date
                        ),
                        "total_assets": get_val(
                            balance,
                            ["Total Assets"],
                            report_date
                        ),
                        "total_debt": get_val(
                            balance,
                            [
                                "Total Debt",
                                "Total Debt And Capital Lease Obligation"
                            ],
                            report_date
                        ),
                        "operating_cash_flow": get_val(
                            cashflow,
                            ["Operating Cash Flow"],
                            report_date
                        ),
                        "capital_expenditure": get_val(
                            cashflow,
                            ["Capital Expenditure", "Purchase Of PPE"],
                            report_date
                        ),
                        "free_cash_flow": get_val(
                            cashflow,
                            ["Free Cash Flow"],
                            report_date
                        ),
                        "stock_repurchased": get_val(
                            cashflow,
                            [
                                "Repurchase Of Capital Stock",
                                "Repurchase Of Stock"
                            ],
                            report_date
                        ),
                        "stock_issued": get_val(
                            cashflow,
                            [
                                "Issuance Of Capital Stock",
                                "Issuance Of Stock"
                            ],
                            report_date
                        ),
                        "ingested_at": ingested_at
                    }
                    all_financials.append(financial_dict)

                if report_dates:
                    print(
                        "  ✅ Financials: "
                        f"{len(report_dates)} kỳ báo cáo"
                    )

            except Exception as e:
                print(f"  ⚠️ Lỗi Financials: {e}")

        except Exception as e:
            print(f"⚠️ Lỗi nghiêm trọng khi xử lý mã {t}: {e}")

        time.sleep(0.5)

    print("\n=== ĐANG GỘP DỮ LIỆU VÀ LƯU LÊN S3 ===")

    # 1. PRICES: Partition theo ngày giao dịch thực tế.
    if all_prices:
        df_prices = pd.concat(all_prices, ignore_index=True)

        df_prices["price_date_dt"] = pd.to_datetime(
            df_prices["price_date"],
            errors="coerce"
        )
        df_prices = df_prices.dropna(subset=["price_date_dt"])

        df_prices = df_prices.drop_duplicates(
            subset=["ticker", "price_date"],
            keep="last"
        )

        for trade_date, df_one_day in df_prices.groupby(
            df_prices["price_date_dt"].dt.normalize()
        ):
            p_yyyy = trade_date.strftime("%Y")
            p_mm = trade_date.strftime("%m")
            p_dd = trade_date.strftime("%d")
            price_prefix = f"{RAW_PREFIX}/{p_yyyy}/{p_mm}/{p_dd}"

            df_save = df_one_day.drop(columns=["price_date_dt"])

            put_df(
                df_save,
                f"{price_prefix}/us_prices_raw.parquet"
            )

    # 2. COMPANY INFO: Partition theo ngày Glue Job chạy.
    if all_info:
        df_info = pd.DataFrame(all_info)

        df_info = df_info.drop_duplicates(
            subset=["ticker", "snapshot_date"],
            keep="last"
        )

        put_df(
            df_info,
            f"{raw_prefix_today}/us_company_info.parquet"
        )

    # 3. FINANCIALS: Partition theo ngày Glue Job chạy.
    if all_financials:
        df_financials = pd.DataFrame(all_financials)

        df_financials = df_financials.drop_duplicates(
            subset=[
                "ticker",
                "report_date",
                "period_type",
                "snapshot_date"
            ],
            keep="last"
        )

        put_df(
            df_financials,
            f"{raw_prefix_today}/us_financials_raw.parquet"
        )

    print("\n🎉 Daily Data Ingestion hoàn tất!")


if __name__ == "__main__":
    try:
        run_yfinance_full_ingestion()

    except Exception as e:
        print("[CRITICAL ERROR]", str(e))
        print(traceback.format_exc())
        raise
````
