Scheduler khởi chạy Glue Python Shell Job hằng ngày. Không cấu hình scheduler chạy cùng lúc transform/publisher vì cần đợi Raw output hoàn chỉnh.

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | EventBridge → Scheduler → **Create schedule** | Tạo recurring schedule |
| 2 | Schedule name | `stock-screener-daily-ingestion` |
| 3 | Schedule pattern | Chọn cron-based schedule |
| 4 | Expression | Ví dụ `cron(0 7 ? * TUE-SAT *)` |
| 5 | Timezone | Chọn `Asia/Ho_Chi_Minh` nếu UI hỗ trợ |
| 6 | Target | Universal target: AWS SDK → Glue → `StartJobRun` |
| 7 | Input payload | Dán JSON bên dưới |
| 8 | Execution role | `EventBridgeSchedulerStockScreenerRole` |
| 9 | Create schedule | Kiểm tra schedule Enabled |
````json
{
  "JobName": "us-stock-daily-ingestion"
}
````
