Tạo ba Bucket tách biệt: Raw, Processed và Static Website. Raw/Processed không public; Website Bucket chỉ public qua CloudFront hoặc Static Website policy sau khi hoàn thiện.

## Tạo Buckets

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Amazon S3 → **Create bucket** | Tạo Raw Bucket |
| 2 | Nhập name | `stock-screener-raw-tên-của-bạn` |
| 3 | Chọn Region | `ap-southeast-1` hoặc Region bạn dùng cho Glue |
| 4 | Giữ Block Public Access | Raw data không public |
| 5 | Lặp lại | Tạo `stock-screener-processed-tên-của-bạn` |
| 6 | Lặp lại | Tạo `stock-screener-web-tên-của-bạn` cho frontend |

## Prefix Contract
````text
# Raw
raw_us_market/YYYY/MM/DD/us_prices_raw.parquet
raw_us_market/YYYY/MM/DD/us_company_info.parquet
raw_us_market/YYYY/MM/DD/us_financials_raw.parquet

# Processed
processed_qmj/as_of_date=YYYY-MM-DD/part-*.snappy.parquet

# Static frontend
index.html
assets/
````
