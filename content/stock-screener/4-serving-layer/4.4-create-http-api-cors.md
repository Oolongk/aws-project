HTTP API expose Lambda Reader thành endpoint để browser gọi. API Gateway route request đến Lambda, Lambda query DynamoDB rồi trả response về client.

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | AWS Console → API Gateway → **Create API** | Chọn HTTP API |
| 2 | Chọn Build | Tại HTTP API |
| 3 | Integration | Chọn Lambda function `stock-screener-reader` |
| 4 | API name | `stock-screener-http-api` |
| 5 | Create | Tạo default stage `$default` hoặc stage riêng |
| 6 | Routes → Create | Method `GET`, route `/scores` |
| 7 | Attach integration | Chọn Lambda reader |
| 8 | CORS | Allowed origins là domain Web Bucket/CloudFront; methods `GET,OPTIONS` |
| 9 | Copy Invoke URL | Dùng cho frontend `API_BASE_URL` |
| 10 | Test | `GET /scores?ticker=AAPL` |

> **Tư duy kiến trúc sư – CORS.** CORS cần cấu hình khi frontend và API ở khác origin. Sau khi deploy production, thay `*` bằng domain website cụ thể.
