#### Tạo file mã nguồn giả lập

Chúng ta sẽ tạo một file Python đóng vai trò là cảm biến thông minh trên xe tải. [cite_start]File này có nhiệm vụ mô phỏng dữ liệu hành trình của 10 xe tải và gửi lên AWS Cloud qua giao thức MQTT. [cite: 1]

1. Mở trình soạn thảo **VS Code**.
2. Tạo một file mới với tên **truck_simulator.py**.
3. Copy và dán đoạn mã dưới đây vào file: 

```python
import time
import json
import random
from AWSIoTPythonSDK.MQTTLib import AWSIoTMQTTClient

# ==========================================
# 🔴 CẤU HÌNH THÔNG SỐ (SẼ CẬP NHẬT Ở CHƯƠNG 2)
# ==========================================
# Endpoint này sẽ lấy từ AWS IoT Core Settings
IOT_ENDPOINT = "YOUR_AWS_IOT_ENDPOINT" 

# Đường dẫn đến các chứng chỉ bảo mật
PATH_TO_ROOT_CA = "certs/AmazonRootCA1.pem"
PATH_TO_CERT = "certs/your-certificate.pem.crt"
PATH_TO_PRIVATE_KEY = "certs/your-private.pem.key"

CLIENT_ID = "Vaccine_Fleet_Simulator"
TOPIC = "telemetry/trucks"

# Tạo danh sách giả lập 10 xe tải với tọa độ gốc
TRUCKS = [{"id": f"T-{100+i}", "lat": 10.7626 + (i * 0.01), "lon": 106.6602 - (i * 0.01)} for i in range(10)]

# ==========================================
# KHỞI TẠO KẾT NỐI MQTT BẢO MẬT
# ==========================================
myMQTTClient = AWSIoTMQTTClient(CLIENT_ID)
myMQTTClient.configureEndpoint(IOT_ENDPOINT, 8883)
myMQTTClient.configureCredentials(PATH_TO_ROOT_CA, PATH_TO_PRIVATE_KEY, PATH_TO_CERT)

print(f"Đang kết nối đến AWS IoT Core...")

# ==========================================
# VÒNG LẶP GỬI DỮ LIỆU (1 GIÂY/LẦN)
# ==========================================
try:
    myMQTTClient.connect()
    print("✅ Kết nối thành công!\n")
    while True:
        for truck in TRUCKS:
            # Sinh dữ liệu ngẫu nhiên cho nhiệt độ và vị trí
            data = {
                "truck_id": truck["id"],
                "temperature": round(random.uniform(2.0, 8.0), 2),
                "latitude": round(truck["lat"] + random.uniform(-0.001, 0.001), 6),
                "longitude": round(truck["lon"] + random.uniform(-0.001, 0.001), 6),
                "timestamp_ms": int(time.time() * 1000)
            }
            
            payload = json.dumps(data)
            myMQTTClient.publish(TOPIC, payload, 1)
            print(f"[{truck['id']}] Đã gửi: {payload}")
            
        time.sleep(1) 

except KeyboardInterrupt:
    print("\n🛑 Đã dừng gửi dữ liệu.")
    myMQTTClient.disconnect()
```
