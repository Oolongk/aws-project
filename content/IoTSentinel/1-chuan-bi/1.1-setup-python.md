#### Thiết lập môi trường Python

Để chạy được mã nguồn giả lập xe tải, máy tính của bạn cần được cài đặt Python và thư viện kết nối IoT của AWS.

1. Kiểm tra phiên bản Python trên máy tính bằng cách mở Terminal/Command Prompt và gõ lệnh:
   `python --version` (Yêu cầu phiên bản 3.9 trở lên).

![Kiểm tra Python](/images/IoTSentinel/1-chuan-bi/python-check.png?featherlight=false&width=90pc)

2. **Cài đặt Python (Dành cho máy chưa có):**
   - **Với Windows:** Truy cập trang [python.org/downloads](https://www.python.org/downloads/), tải bộ cài đặt mới nhất. 
     > **Lưu ý quan trọng:** Ở màn hình cài đặt đầu tiên, BẮT BUỘC phải tích chọn ô **"Add Python.exe to PATH"** trước khi nhấn Install Now. Điều này giúp Terminal nhận diện được lệnh `python`.
   - **Với macOS (Dùng Homebrew):**
     ```bash
     brew install python
     ```
   - **Với Linux (Ubuntu/Debian):**
     ```bash
     sudo apt update && sudo apt install python3 python3-pip -y
     ```

3. Cài đặt thư viện **AWSIoTPythonSDK**. Đây là thư viện quan trọng nhất để thực hiện giao tiếp MQTT.
   Chạy lệnh:
   
   ```bash
   pip install AWSIoTPythonSDK
   ```
