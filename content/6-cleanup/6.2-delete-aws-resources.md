Xóa resource theo dependency từ public entry point về data storage.

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | EventBridge Scheduler | Disable hoặc Delete schedule trước |
| 2 | API Gateway | Delete HTTP API |
| 3 | Lambda Reader | Delete `stock-screener-reader` sau API Gateway |
| 4 | S3 Trigger | Xóa S3 event notification hoặc delete Lambda Loader sẽ được trigger remove |
| 5 | Lambda QMJ Loader | Delete `qmj-s3-to-dynamodb-loader` |
| 6 | DynamoDB | Delete table `qmj_screener` nếu không cần demo history |
| 7 | Glue Jobs | Delete Daily Ingestion và QMJ Transform jobs |
| 8 | S3 Web Bucket | Empty bucket rồi Delete |
| 9 | S3 Processed Bucket | Xác nhận archive trước rồi Empty/Delete |
| 10 | S3 Raw Bucket | Xác nhận archive trước rồi Empty/Delete |
| 11 | IAM Inline Policies/Roles | Detach/Delete Role nếu dedicated cho project |
