Kinesis Data Firehose đóng vai trò là trạm tiếp nhận dữ liệu streaming và thực hiện chuyển đổi định dạng (JSON sang Parquet) ngay trên luồng dữ liệu (on-the-fly).

#### Hướng dẫn triển khai từng bước (Step-by-step)

| Bước | Hành động | Chi tiết kỹ thuật |
| :--- | :--- | :--- |
| **1** | Truy cập AWS Console | Tìm kiếm **Kinesis** -> chọn **Data Firehose**. |
| **2** | Tạo Stream | Bấm **Create delivery stream**. |
| **3** | Cấu hình Source | Chọn Source là **Direct PUT**. |
| **4** | Cấu hình Destination | Chọn Destination là **Amazon S3**. |
| **5** | Bật tính năng nén | Tìm mục **Transform and convert records**, chọn **Enable** ở ô *Convert record format*. |
| **6** | Lựa chọn Format | Chọn định dạng Output là **Apache Parquet**. |
| **7** | Trỏ về AWS Glue Catalog | Ở phần Schema configuration, chọn đúng **Database** (`clickstream_db`) và **Table** mà Crawler vừa sinh ra ở bài 1.2. |
| **8** | Cấu hình S3 đích | Tại mục Destination settings, chọn Bucket S3 của bạn và nhập Prefix là `data/`. Bấm Create. |

> **Lưu ý quan trọng:** Tại mục **Destination settings**, hãy đảm bảo bạn chọn đúng bucket S3 đã tạo ở Chương 1. Tại phần **Prefix**, hãy nhập `data/` để Firehose tự động gom dữ liệu vào thư mục này.

---

#### Kiểm tra kết quả
Sau khi bấm **Create delivery stream**, bạn hãy quay lại giao diện S3. Nếu thấy các thư mục được tạo tự động dạng `/year/month/day/hour/`, điều đó có nghĩa là hệ thống đã sẵn sàng tiếp nhận dữ liệu!
