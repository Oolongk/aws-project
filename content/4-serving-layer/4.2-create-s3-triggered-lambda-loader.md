`qmj_transform.py` ghi Parquet vào `processed_qmj/as_of_date=YYYY-MM-DD/`. Bài này dùng Lambda QMJ Loader của bạn để nhận S3 `ObjectCreated` event, đọc part Parquet bằng `awswrangler` và batch write sang DynamoDB.

Lambda này có hai mode:

| Mode | Cách gọi | Mục đích |
|---|---|---|
| `s3_trigger` | S3 tự gửi event khi có `.parquet` mới | Nạp incremental output mới từ Glue PySpark |
| `backfill` | Lambda Test Event hoặc Invoke API | Quét toàn bộ `processed_qmj/` và nạp historical QMJ |

> **Tư duy kiến trúc sư – source boundary.** Đây không phải Glue Publisher Job. S3 là event source, Lambda là compute layer nạp Serving Store, còn DynamoDB là read-optimized projection cho website.

# Chuẩn bị deployment package

Source của bạn import `awswrangler`, `pandas` và `pyarrow` dependency. Lambda runtime mặc định không tự có toàn bộ package này, vì vậy bạn cần gắn Lambda Layer hoặc upload deployment package tương thích với Python runtime và architecture đang dùng.

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Mở AWS CloudShell hoặc môi trường build tương thích Lambda | Dùng để tạo package Linux-compatible |
| 2 | Tạo thư mục package | Package phải chứa `lambda_function.py` và dependencies |
| 3 | Cài dependencies | Cài `awswrangler`, `pandas`, `pyarrow` vào package hoặc Layer |
| 4 | Copy source Lambda | Lưu source dưới tên `lambda_function.py` |
| 5 | Zip package | Upload zip vào Lambda ở bước kế tiếp |

````bash
# Chạy trong môi trường build tương thích Lambda.
mkdir qmj-loader-package
cd qmj-loader-package

# Cài dependency cùng source Lambda vào package.
python -m pip install --target . awswrangler pandas pyarrow
cp ../lambda_qmj_loader.py lambda_function.py

# Tạo file upload cho Lambda.
zip -r ../qmj-loader.zip .
````

# Tạo S3 Trigger

| Bước | Hành động | Chi tiết kỹ thuật |
|---|---|---|
| 1 | Lambda → Function overview → **Add trigger** | Chọn trigger source `S3` |
| 2 | Bucket | Chọn `stock-screener-processed-tên-của-bạn` |
| 3 | Event type | Chọn `All object create events` |
| 4 | Prefix | Nhập `processed_qmj/` |
| 5 | Suffix | Nhập `.parquet` |
| 6 | Recursive invocation | Bật acknowledgement nếu Console hiển thị | Lambda không ghi object lại S3 nên không tạo loop |
| 7 | Chọn **Add** | S3 event notification sẽ invoke Lambda |

Khi Glue PySpark ghi output như dưới đây, S3 trigger chỉ chuyển file `part-...parquet`, còn `_SUCCESS` bị bỏ qua do suffix filter và code Lambda.

````text
processed_qmj/
└── as_of_date=2026-07-06/
    ├── part-00000-....snappy.parquet   ← Lambda trigger
    └── _SUCCESS                         ← ignored
````

# Lambda Loader source code

Dòng `import json` được thêm vì source cần `json.dumps(response)` khi có `failed_files`.

````python
import json
import os
import re
import math
import boto3
import urllib.parse
import awswrangler as wr
import pandas as pd

from datetime import date, datetime
from decimal import Decimal
from numbers import Integral, Real

TABLE_NAME = os.environ.get("TABLE_NAME", "qmj_screener")
PROCESSED_BUCKET = os.environ.get(
    "PROCESSED_BUCKET",
    "stock-screener-processed-tên-của-bạn"
)
PROCESSED_PREFIX = os.environ.get(
    "PROCESSED_PREFIX",
    "processed_qmj/"
)

dynamodb = boto3.resource("dynamodb")
s3 = boto3.client("s3")
table = dynamodb.Table(TABLE_NAME)


def to_dynamodb_value(value):
    # Chuẩn hóa Pandas/NumPy value về type DynamoDB chấp nhận.
    if value is None:
        return None

    try:
        if bool(pd.isna(value)):
            return None
    except Exception:
        pass

    if isinstance(value, (datetime, date)):
        return value.strftime("%Y-%m-%d")

    if isinstance(value, bool):
        return value

    if isinstance(value, Decimal):
        return value

    if isinstance(value, Integral):
        return Decimal(str(int(value)))

    if isinstance(value, Real):
        number = float(value)
        if not math.isfinite(number):
            return None
        return Decimal(str(number))

    if isinstance(value, str):
        return value

    return str(value)


def extract_as_of_date(s3_key):
    # Lấy partition date từ processed_qmj/as_of_date=YYYY-MM-DD/.
    match = re.search(
        r"(?:^|/)as_of_date=(\d{4}-\d{2}-\d{2})(?:/|$)",
        s3_key
    )

    if not match:
        raise ValueError(f"Không tìm thấy as_of_date trong key: {s3_key}")

    as_of_date = match.group(1)
    datetime.strptime(as_of_date, "%Y-%m-%d")
    return as_of_date


def load_parquet_to_dynamodb(bucket, key):
    as_of_date = extract_as_of_date(key)
    s3_path = f"s3://{bucket}/{key}"
    df = wr.s3.read_parquet(path=s3_path)

    if df.empty:
        return 0, 0

    if "ticker" not in df.columns:
        raise ValueError(f"File không có cột ticker: {s3_path}")

    loaded_rows = 0
    skipped_rows = 0
    columns = list(df.columns)

    # Overwrite cùng ticker + snapshot_date để retry event không tạo duplicate logical item.
    with table.batch_writer(
        overwrite_by_pkeys=["ticker", "snapshot_date"]
    ) as batch:
        for values in df.itertuples(index=False, name=None):
            item = {}

            for column, value in zip(columns, values):
                clean_value = to_dynamodb_value(value)
                if clean_value is not None:
                    item[column] = clean_value

            ticker = str(item.get("ticker", "")).strip().upper()
            if not ticker:
                skipped_rows += 1
                continue

            item["ticker"] = ticker
            item["snapshot_date"] = as_of_date
            item["as_of_date"] = as_of_date
            item["source_s3_key"] = key

            batch.put_item(Item=item)
            loaded_rows += 1

    return loaded_rows, skipped_rows


def get_existing_parquet_files(bucket, prefix):
    keys = []
    paginator = s3.get_paginator("list_objects_v2")

    for page in paginator.paginate(Bucket=bucket, Prefix=prefix):
        for obj in page.get("Contents", []):
            key = obj["Key"]
            if key.endswith(".parquet"):
                keys.append(key)

    return sorted(keys)


def process_files(bucket, keys):
    processed_files = 0
    loaded_rows = 0
    skipped_rows = 0
    failed_files = []

    for key in keys:
        try:
            loaded, skipped = load_parquet_to_dynamodb(bucket, key)
            processed_files += 1
            loaded_rows += loaded
            skipped_rows += skipped
        except Exception as error:
            failed_files.append({"key": key, "error": str(error)})

    return {
        "processed_files": processed_files,
        "loaded_rows": loaded_rows,
        "skipped_rows": skipped_rows,
        "failed_files": failed_files
    }


def lambda_handler(event, context):
    event = event or {}

    # Manual backfill mode: quét toàn bộ QMJ Parquet trong Processed Zone.
    if event.get("mode") == "backfill":
        bucket = event.get("bucket", PROCESSED_BUCKET)
        prefix = event.get("prefix", PROCESSED_PREFIX)
        keys = get_existing_parquet_files(bucket, prefix)

        if not keys:
            return {
                "mode": "backfill",
                "processed_files": 0,
                "loaded_rows": 0,
                "skipped_rows": 0,
                "failed_files": []
            }

        result = process_files(bucket, keys)
        result["mode"] = "backfill"
        result["total_parquet_files_found"] = len(keys)
        return result

    # S3 trigger mode: lọc event ObjectCreated trong prefix Processed QMJ.
    keys_by_bucket = {}

    for record in event.get("Records", []):
        if record.get("eventSource") != "aws:s3":
            continue

        if not record.get("eventName", "").startswith("ObjectCreated:"):
            continue

        bucket = record["s3"]["bucket"]["name"]
        key = urllib.parse.unquote_plus(record["s3"]["object"]["key"])

        if not key.startswith(PROCESSED_PREFIX) or not key.endswith(".parquet"):
            continue

        keys_by_bucket.setdefault(bucket, []).append(key)

    total_processed_files = 0
    total_loaded_rows = 0
    total_skipped_rows = 0
    failed_files = []

    for bucket, keys in keys_by_bucket.items():
        result = process_files(bucket, keys)
        total_processed_files += result["processed_files"]
        total_loaded_rows += result["loaded_rows"]
        total_skipped_rows += result["skipped_rows"]
        failed_files.extend(result["failed_files"])

    response = {
        "mode": "s3_trigger",
        "processed_files": total_processed_files,
        "loaded_rows": total_loaded_rows,
        "skipped_rows": total_skipped_rows,
        "failed_files": failed_files
    }

    if failed_files:
        raise Exception(json.dumps(response))

    return response
````

> **Tư duy kiến trúc sư – event delivery.** Lambda có thể nhận lại một event khi S3 retry. Loader xử lý idempotent ở business key `ticker + snapshot_date`; event duplicate không làm xuất hiện hai item logical trong DynamoDB.
